# 图像信息隐藏与分析(LSB & DCT Analysis)

本项目是一个基于 Python 的图像处理工具，主要演示了**最低有效位 (LSB)** 信息隐藏技术、图像质量评价（PSNR）、**位平面分解**以及**离散余弦变换 (DCT)** 的频域分析。

## 🚀 功能特性

1.  **图像预处理**：自动将输入图像转换为灰度图并缩放至 512x512 分辨率。
2.  **LSB 信息隐藏**：
    *   **嵌入 (Encoding)**：将字符串信息转换为二进制，并按像素嵌入到图像的 LSB 位。
    *   **提取 (Decoding)**：从含有信息的图像中提取出隐藏的原始字符串。
3.  **质量评估**：计算原始图像与嵌入信息后图像的 **PSNR（峰值信噪比）**，量化评估图像失真程度。
4.  **位平面分析**：将图像分解为 8 个位平面，直观展示图像不同位的重要性（高位承载轮廓，低位承载细节/噪声）。
5.  **DCT 变换演示**：
    *   对图像进行 DCT 变换。
    *   模拟频域失真（通过置零高频系数）。
    *   进行逆 DCT 变换（IDCT）重构图像，展示频域过滤对视觉效果的影响。

## 📖 使用指南

1.  **准备素材**：准备一张名为 `bupt.bmp` 的图像放在脚本同一目录下。
2.  **运行脚本**：
    ```bash
    python main.py
    ```
3.  **查看结果**：
    *   控制台将输出提取出的秘密信息以及 **PSNR 值**。
    *   脚本会依次弹出 Matplotlib 窗口，展示各个**位平面**的效果以及 **DCT 变换/重构** 的对比图。

## 🔍 技术细节

### LSB (Least Significant Bit)
脚本通过修改像素值的最后一位来存储二进制数据。由于修改极小（灰度值仅改变 0 或 1），肉眼几乎无法察觉。

### PSNR (Peak Signal-to-Noise Ratio)
用于衡量图像重建质量。
*   PSNR > 40dB：图像质量极好（基本等同于原图）。
*   30dB - 40dB：图像质量较好（肉眼察觉不到或很难察觉失真）。

### Bit-Plane Decomposition
图像由 8 个位平面组成。从第 1 位（最低位）到第 8 位（最高位）。本项目展示了去掉某一位平面后图像的变化，帮助理解信息隐藏对图像的影响。

### DCT (Discrete Cosine Transform)
在 JPEG 压缩等领域广泛应用。脚本演示了通过滤除高频分量来模拟图像压缩后的质量损失。

## ⚠️ 注意事项

*   代码默认处理的是灰度化后的 3 通道逻辑（通过 `convert('RGB')` 处理），在纯灰度单通道上嵌入时逻辑略有不同。
*   由于 LSB 比较脆弱，对图像进行任何形式的压缩（如转存为 JPG）或缩放都会导致隐藏信息丢失。
