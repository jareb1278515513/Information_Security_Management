# CNN数据投毒攻击与

👤 **提交者：** 肖熠

## 1. 项目简介
实验基于 PyTorch 深度学习框架，针对 MNIST 手写数字数据集构建了适配的 AlexNet 卷积神经网络模型。并在训练数据加载阶段，内嵌了可控的**“数据投毒（Poisoning Attack）”**机制。通过在训练集中故意引入不同比例、不同规律的错误标签数据（即投毒样本），观察并分析模型在测试集上的分类准确率与损失函数（Loss）的异常变化。

## 2. 核心功能
本代码共计 160 余行，包含了完整的深度学习攻防测试流水线：
*   **适配性网络构建：** 自定义修改了 AlexNet 的网络层数与卷积核填充，使其能够处理 28x28 尺寸的小特征图。
*   **恶意数据注入逻辑：** 编写了自定义的 `fetch_datasets` 函数，能够按指定比例（如 50%、100%）和特定规律篡改训练集标签，生成“投毒训练集”。
*   **对抗训练与监控：** 实现了完整的模型迭代训练过程，并实时记录干净样本与被投毒样本对模型收敛状态（Loss）的影响。
*   **可视化分析模块：** 依赖 Matplotlib 将训练过程中的准确率震荡、Loss 变化绘制成折线图，并能直接渲染展示被模型错误分类的样本图像。

## 3. 运行环境依赖
*   Python 3.8+
*   PyTorch 1.7.0+ & torchvision
*   numpy, matplotlib
